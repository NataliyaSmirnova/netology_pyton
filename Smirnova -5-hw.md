# Задание 2

#### Создайте диагональную матрицу с элементами от N до 0. Посчитайте сумму ее значений на диагонали.


```python
n = int(input('Введите число элементов '))
matrix = np.diag(np.arange (n-1,-1,-1))
sum(sum(matrix))
```

